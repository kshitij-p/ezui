## <HeaderLink href="#usage" id="usage">Usage</HeaderLink>

If options are **objects**

<CodeCopyBlock>
  ```tsx
  <Autocomplete options={options} label="label">
    <AutocompleteTrigger>
      <AutocompleteValue />
    </AutocompleteTrigger>
    <AutocompleteContent>
      {frameworks.map((option) => (
        <AutocompleteItem key={options.value} value={option}>
          {framework.label}
        </AutocompleteItem>
      ))}
    </AutocompleteContent>
  </Autocomplete>
  ```
</CodeCopyBlock>

If options are **strings**

<CodeBlock id="usage">
  ```tsx
  <Autocomplete options={options}>
    <AutocompleteTrigger>
  ```

</CodeBlock>
