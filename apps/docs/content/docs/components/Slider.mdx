## <HeaderLink href="#usage" id="usage">Usage</HeaderLink>

### For simple cases

<CodeCopyBlock>
  ```tsx
  <Slider defaultValue={[50]} max={100} step={1} />
  //Range Slider
  <RangeSlider defaultValue={[25, 50]} max={100} step={1} />
  ```
</CodeCopyBlock>

### For when you need more control

<CodeCopyBlock>
  ```tsx
  <SliderRoot defaultValue={[50]} max={100} step={1}>
    <SliderTrack>
      <SliderRange />
    </SliderTrack>
    <SliderThumb />
  </SliderRoot>
  //Range Slider
  <SliderRoot defaultValue={[25, 50]} max={100} step={1}>
    <SliderTrack>
      <SliderRange />
    </SliderTrack>
    <SliderThumb />
    <SliderThumb />
  </SliderRoot>
  ```
</CodeCopyBlock>
